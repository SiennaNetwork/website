WRAP

IBC Wrapping
SiennaSwap features a built-in wrapping tool that allows users to convert their native Inter-Blockchain Communication protocol (IBC) tokens into secret tokens. To do this, users must first perform an IBC deposit directly on SiennaSwap.

Once they have deposited their native IBC tokens, they can then use the IBC wrapping tool to convert them into secret tokens, at which point they can use them to interact with the SiennaSwap app. Users can convert their secret tokens back to their native versions and withdraw them in a permission-less manner.


Supported Tokens

SCRT ⇄ sSCRT
ATOM ⇄ sATOM
Luna ⇄ sLuna
UST ⇄ sUST
OSMO ⇄ sOSMO
DVPN ⇄ sDVPN


IBC Deposits/Withdrawals: app.sienna.network/wrap/ibc
